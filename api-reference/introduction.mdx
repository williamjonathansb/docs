## Códigos de resposta

Melifresh usa códigos HTTP padrão para indicar o sucesso ou falha de suas solicitações.

Em geral, códigos HTTP 2xx correspondem a sucesso, códigos 4xx são para falhas relacionadas ao usuário e códigos 5xx são para problemas de infraestrutura.

| Status   | Descrição                             |
| -------- | ------------------------------------- |
| `200`    | Solicitação bem-sucedida.                     |
| `422`    | Verifique se os parâmetros estavam corretos.  |
| `404`    | O recurso não foi encontrado.                 |
| `409`    | O recurso já existe                           |
| `5xx`    | Indica um erro com servidores do melifresh.   |
